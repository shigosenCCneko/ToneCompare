# ToneCompare
## 音色比較ツール

２つの音色を波形、FFT,TFFTで比較するツールです
Recordボタンで２秒間録音します
FFT標示の間のプルダウンメニューで入力デバイスが指定出来ます

### ボタン

Record  ２秒間録音  
Play    再生

### スライダー
MAG     波形標示の倍率変更  
FFT Pos FFT解析開始位置  

###　ラジオボタン  
FFT sampling Rate   解析レート  
Real Time Wave      リアルタイム波形表示(ch2のみ)  
Real TIme FFT       リアルタイムFFT表示(ch2のみ)  

FFT Pos Sync 1ch    2chと1chのFF Posを同期させる
